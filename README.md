# Pingo

A Python project that measures network quality using ping, created by a programming beginner.

# Funtions
- response time calculation
- packet loss analysis
- classification of the connection based on response time and packet loss

# How to use
- git clone https://github.com/Far-Net/Pingo.git or download the zip file
- use: pip install -r requirements.txt
- and use: Python Pingo.py
